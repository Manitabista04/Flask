# Flask Web App

A lightweight **Flask** project structured for easy development and cloud deployment.

### Structure
* `app.py`: Main server logic and routes.
* `templates/`: HTML frontend files.
* `requirements.txt`: Python dependencies.
* `Procfile`: Deployment configuration (Heroku).

### Quick Start
1. **Install:** `pip install -r requirements.txt`
2. **Run:** `python app.py`
3. **Access:** `http://127.0.0.1:5000/`
